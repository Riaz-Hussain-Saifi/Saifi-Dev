## Setup Instructions
To run the Next.js portfolio locally:
1. **Clone the Repo**:
   ```bash
   git clone https://github.com/Riaz-Hussain-Saifi/Saifi-Dev.git
   cd Saifi-Dev
   ```
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run Development Server**:
   ```bash
   npm run dev
   ```
   Visit `http://localhost:3000` in your browser.
